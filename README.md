PlayBound is a full-stack web platform designed to help individuals find suitable partners for recreational sports and indoor games. Built to reduce the friction of coordinating casual games, PlayBound connects local communities through a beautiful, Studio Ghibli-inspired user interface.

---

## 🎯 Problem Statement
Many individuals struggle to find partners for recreational games due to a lack of awareness about nearby players, difficulty coordinating times, and limited access to local community networks. PlayBound solves this by providing a localized, intuitive matchmaking platform.

## ✨ Key Features

* **Advanced Player Discovery:** A robust 4-part search engine allowing users to find partners based on **Game Type**, **City/Region**, **Availability**, and **Skill Level**.
* **Seamless Matchmaking:** Users can send, receive, accept, or decline play requests with a real-time status dashboard.
* **Studio Ghibli-Inspired UI:** A custom, tranquil user interface featuring soft emerald palettes, custom CSS animations, and looping atmospheric backgrounds for an elevated UX.
* **Secure Authentication:** Full JWT (JSON Web Token) integration for secure user registration, login, and protected routes.
* **Admin Control Panel:** A protected "God Mode" dashboard for administrators to monitor the platform and manage/ban users to maintain community safety.

## 🛠️ Technology Stack

**Frontend:**
* React.js (Vite)
* Tailwind CSS (v4)
* React Router DOM
* Axios

**Backend:**
* Node.js & Express.js
* MongoDB & Mongoose
* JSON Web Tokens (JWT) & bcryptjs (Security)
